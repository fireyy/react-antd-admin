## Getting Started

Just clone the repo and install the necessary node modules:

```shell
$ git clone https://github.com/fireyy/react-antd-admin
$ cd react-antd-admin
$ npm install
```

## Run Dev

```shell
$ npm run dev
```

## Run test spec

```shell
$ npm run test
```

## Build

```shell
$ npm run build
```
